## Installation
run the command
```bash
git clone https://github.com/vadykoo/test-football.git
composer install
```
create .env file and database and run the commands

```bash
php artisan migrate -seed
php artisan serve
```

Open the site and login with

```bash
Login: super@admin.com
Password: secret
```
or
```bash
Login: club@admin.com
Password: secret
```
